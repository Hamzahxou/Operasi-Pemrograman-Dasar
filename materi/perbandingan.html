Operator Perbandingan

Digunakan untuk membandingkan dua nilai dan menghasilkan nilai boolean (true atau false).

Operator Penjelasan Contoh
== Sama dengan 5 == 5 (true)
=== Identik (tipe & nilai) 5 === "5" (false)
!= Tidak sama dengan 5 != 3 (true)
> Lebih besar 10 > 5 (true)
<= Lebih kecil sama dengan 5 <= 5 (true)

Penting: == vs ===

Banyak pemula bingung dengan dua ini.
== (Equal) hanya membandingkan nilainya saja.
=== (Strict Equal) membandingkan nilai DAN tipe datanya. Dalam praktik industri, disarankan untuk selalu menggunakan === agar hasil perbandingan lebih akurat dan terhindar dari *bug*.

// Contoh Perbandingan
let a = 10;
console.log(a == "10"); // true (karena nilai sama)
console.log(a === "10"); // false (karena tipe data beda, angka vs string)
