Operator Penugasan

Operator ini digunakan untuk memberikan nilai ke dalam suatu variabel.

Operator Contoh Sama dengan
= x = 10 x = 10
+= x += 5 x = x + 5
-= x -= 2 x = x - 2
*= x *= 3 x = x * 3
/= x /= 2 x = x / 2

Mengapa pakai operator penugasan singkat?

Penggunaan operator seperti += adalah cara yang lebih ringkas (shorthand) untuk memperbarui nilai variabel. Ini sangat sering digunakan dalam perulangan (*looping*) atau saat menambahkan skor pada game sederhana.

// Contoh Skor Game
let skor = 0;
skor += 10; // Skor sekarang menjadi 10
skor += 5; // Skor sekarang menjadi 15
console.log(skor);
