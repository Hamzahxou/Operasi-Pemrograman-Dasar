Increment & Decrement

Operator khusus untuk menambah atau mengurangi nilai variabel sebanyak satu satuan (+1 atau -1).

Increment (++)

Menambahkan 1 ke nilai variabel.

x++; // x = x + 1

Decrement (--)

Mengurangi 1 dari nilai variabel.

x--; // x = x - 1

Perbedaan Posisi (Prefix vs Postfix)

Penting untuk membedakan antara ++x (Prefix) dan x++ (Postfix).

  • Postfix (x++): Nilai digunakan dulu, baru kemudian ditambah.
  • Prefix (++x): Nilai ditambah dulu, baru kemudian digunakan.
// Contoh Postfix
let x = 5;
console.log(x++); // Output: 5 (karena ditampilkan dulu)
console.log(x); // Output: 6 (sudah ditambah)
