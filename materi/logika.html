Operator Logika

Digunakan untuk menggabungkan dua atau lebih kondisi logika.

AND (&&)

Hasilnya TRUE jika semua kondisi bernilai benar.

OR (||)

Hasilnya TRUE jika salah satu kondisi bernilai benar.

NOT (!)

Membalikkan nilai logika (TRUE jadi FALSE).

Tabel Kebenaran (AND)

Kondisi A Kondisi B A && B
TRUE TRUE TRUE
TRUE FALSE FALSE
FALSE FALSE FALSE
// Contoh: Login Sistem
let user = "admin";
let pass = "12345";

if (user === "admin" && pass === "12345") {
  console.log("Login Berhasil!");
}
